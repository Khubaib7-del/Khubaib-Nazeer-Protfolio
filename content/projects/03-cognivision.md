# Project: CogniVision

- Name: CogniVision
- Pitch: AI-powered live student attention and cheating-behavior detection for exams/proctoring —
  dual-detector pipeline (EfficientNet-B2 attention classifier + custom YOLOv8 behavior detector)
  with a real-time FastAPI dashboard.
- Stack: Python, PyTorch (EfficientNet-B2), YOLOv8, FastAPI
- Highlights: 90.10% validation accuracy (attention classifier), 37.3% mAP@0.5 (behavior detector)
